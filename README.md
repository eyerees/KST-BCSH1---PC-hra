# Fandango (Unity Project)

Projekt 2D RPG z ptačí perspektivy vytvořený v enginu Unity.

## 📅 Stav projektu (k 15. 04. 2026)
* **Systémy:** Main Menu, UI, Saving, Dialogue, Quest, Combat, Sound.
* **Obsah:** 3 lokace.

---

## 🛠️ Technické specifikace

| Sloupec 1 (Kategorie) | Sloupec 2 (Detail) |
| :--- | :--- |
| **Engine** | Unity 6 (6000.3.10f1) |
| **Skriptování** | C# (VS Code) |
| **Fyzika** | Rigidbody2D |
| **Grafika** | 2D Top-Down Sprite Sheet + Tilemap |
